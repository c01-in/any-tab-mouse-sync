# Contributing

Thanks for your interest in improving Any Tab Mouse Sync.

## Development Setup

1. Open Chrome at `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this project folder

## Pull Request Notes

- Keep changes focused and small
- Explain behavior changes in the PR description
- Update `README.md` if user-facing behavior or permissions change
- Do not include generated zip files from `dist/` in PRs

## Reporting Bugs

When filing an issue, include:

- Repro steps
- Expected behavior
- Actual behavior
- Chrome version and OS
- Whether the page uses Shadow DOM / custom context menus
